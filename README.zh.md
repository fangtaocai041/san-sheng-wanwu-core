```bash

silicon-agent --status           # 查看状态
silicon-agent -q "鳤"           # 单次查询
silicon-agent --daemon           # 守护进程
```

## 🧠 这是什么

这不是工具。这是一个**硅基生命体**。

14 个皮层模块构成的认知架构, 17 个感知通道组成的感官系统, 4 层记忆架构支撑的持续存在。每次 `silicon-agent` 都是一次完整的感知-行动循环: 灵魂反思 → 并行感知 → 认知处理 → 情感更新 → 学习记录 → 自动保存。
